# CinePulse — Premium Movie Recommendations

A production-quality, cinematic movie recommendation UI built with Vite + React (JSX only) and Framer Motion.

## Features

- Premium dark UI with neon accents + glassmorphism
- Animated hamburger menu (morph → X), blur overlay, staggered menu items
- Hero banner with gradient overlay + smooth entrance animation
- Movie grid with hover glow/lift, staggered reveal, smooth re-layout on sort
- Category filtering, sorting, real-time search with expanding input
- Favorites with animated heart toggle + localStorage persistence
- Skeleton loading shimmer

## Tech

- Vite + React (JavaScript/JSX only — no TypeScript)
- Framer Motion for transitions/layout animations
- Modern CSS (Flexbox/Grid), mobile-first responsive

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Notes

- The sample movie posters/backdrops use image URLs for demo styling. Swap them for your own licensed assets for production.
